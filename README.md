# EDT DCSQueryMenuEx

Плагин для EDT, добавляющий пункт **«Редактор запроса...»** в контекстное меню редактора схемы СКД, позволяя открывать окно редактора запроса в обход конструктора.

## Установка

- **Через GitHub:** Help → Install New Software → Add → `https://timkonzt.github.io/EDT-DCSQueryMenuEx/`
- **Из ZIP-архива:** Help → Install New Software → Add → Archive → указать `edt_dcs_querymenyex_<версия>.zip` из [релизов](https://github.com/TimkoNZT/EDT-DCSQueryMenuEx/releases)
